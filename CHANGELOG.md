# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-24

### Added
- Initial release of the TodoList component
- TypeScript support with full type definitions
- Clean, modern UI with light and dark theme support
- Comprehensive feature set:
  - Add, edit, delete, and toggle todo items
  - Priority levels (low, medium, high)
  - Categories and tags
  - Due dates with overdue indicators
  - Assignment to users
  - Filtering (all, active, completed)
  - Sorting by multiple criteria
  - Inline editing
  - Search/filter functionality
- Responsive design for mobile and desktop
- Full accessibility support
- Storybook integration for component documentation
- Jest tests for component reliability
- ESLint configuration for code quality
- Rollup build system for optimized bundle
- Comprehensive documentation and examples
